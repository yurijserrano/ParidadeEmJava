Iniciando o Simulador

 O que é o Simulador?
O JPACS (Java Processor and Cache Simulator) é um programa que simula a memória cache de um computador. O próprio JPACS possui uma memória ram, e um processador virtuais para auxiliar na simulação do cache.

 Menu do JPACS
O Menu do simulador possui várias opções que podem chegar a confundir o usuário, para evitar este problema vamos descrever com detalhes o que cada opção do menu faz:

 Arquivo:
      Novo - cria novo arquivo de Acesso direto ou Assembly. Para saber os comandos Assembly vá ao tópico Utilizando o Assembly.
      Abrir - abre arquivo escolhido em nova janela. Atenção: O Simulador diferencia arquivos de Acesso direto e Assembly simplesmente pela extensão do arquivo.
      Salvar - salva janela ativa para arquivo. Se todas janelas de Acesso Direto e Assembly estiverem inativas, o programa irá salvar uma delas. Atenção: Por favor coloque sempre a extensão no nome do arquivo para o bom funcionamento do programa.
      Sair - saída do simulador.

 Simulação:
      Relâmpago - simula a janela de Acesso Direto Ativa ou inicia o processador caso janela de Acesso Direto não exista ou estaja inativa. Esta simulação apenas exibe os resultados quando acabar a simulação.
      Passo a Passo - simula a janela de Acesso Direto Ativa ou inicia o processador caso janela de Acesso Direto não exista ou estaja inativa. Esta simulação simula uma instrução de Assembly (ou uma linha de Acesso Direto) por vez.
      Salvar Relatório - salva o conteúdo da janela Relatório de Simulação no arquivo escolhido. Pode ser salvo tanto em formato Texto quanto em Xml. Atenção: Por favor coloque sempre a extensão no nome do arquivo para o bom funcionamento do programa.
      Limpar - Limpa todos os componentes do simulador para que uma nova simulação possa ser feita.

 Configurações:
      Memórias - configura a memória principal e cache. Pode-se configurar o tamanho, tempo de acesso, entre outras opções.
      Internacional - configura o idioma do JPACS.

 Janela:
      Memória Principal - exibe ou oculta a janela contendo a memória principal.
      Memória Cache - exibe ou oculta a janela contendo a memória cache.
      Processador - exibe ou oculta a janela contendo as informações do processador.
      Relatório de Simulação - exibe ou oculta a janela contendo o relatório de simulação.
      Outras Janelas - serão criados menus para cada janela de assembly e acesso direto que o simulador estiver exibindo.

 Ajuda:
      Tópicos de Ajuda - ajuda do simulador.
      Comandos Assembly - janela para rápida consulta dos comandos Assembly.
      Sobre - os desenvolvedores desse software.
 

 Acesso Direto
Um objeto de Acesso Direto é bastante simples. Você deve escrever a posicão de memória a ser acessada, um espaço e uma das letras l ou e (leitura ou escrita, caso outra letra seja especificada, o simulador irá realizar uma leitura). Exemplo:

100 e
200 e
300 l
100 l
